# 活动策划与管理系统 - 需求文档

## 1. 项目概述

### 1.1 项目背景

本系统是一个综合性的活动策划与管理系统，旨在为文化传媒公司提供完整的活动管理解决方案，涵盖活动申请、人员安排、物料管理、薪资管理、财务统计等核心业务流程。

### 1.2 目标用户

- **普通用户**：申请活动、查看活动进度、支付费用
- **业务管理员**：审批活动申请、安排人员、管理活动进度
- **仓库管理员**：管理物料出入库、审批物料申请
- **财务人员**：管理薪资发放、统计财务收支
- **系统管理员**：用户管理、权限管理、系统配置

## 2. 功能需求

### 2.1 用户管理模块

#### 2.1.1 用户注册

- **功能描述**：新用户注册账号
- **输入项**：用户名、密码、角色（用户/员工）
- **业务规则**：
  - 用户名唯一性校验
  - 员工注册需额外填写职位、城市、姓名、电话、性别

#### 2.1.2 用户登录

- **功能描述**：用户登录系统
- **输入项**：用户名、密码、角色
- **业务规则**：
  - 验证用户名、密码、角色匹配
  - 检查账号状态（正常/封禁）
  - 登录成功后保存用户信息到本地存储

#### 2.1.3 用户信息管理

- **功能描述**：查看和修改用户信息
- **功能项**：
  - 查看个人资料（用户名、姓名、电话、性别、职位等）
  - 修改个人信息（电话、性别、姓名）
  - 修改密码（需验证原密码）

#### 2.1.4 用户权限管理

- **功能描述**：管理员管理用户账号状态
- **功能项**：
  - 查看所有用户列表
  - 封禁/解封用户账号
  - 按用户名、ID、姓名搜索用户

---

### 2.2 活动管理模块

#### 2.2.1 活动申请

- **功能描述**：用户提交活动申请
- **输入项**：
  - 活动地点
  - 活动类型（公司年会、招聘会等）
  - 活动开始时间
  - 活动结束时间
  - 参与人数
  - 活动预算
  - 需求描述
- **业务规则**：
  - 自动设置活动状态为"待处理"（0）
  - 关联申请人用户ID

#### 2.2.2 活动审批流程

活动状态流转：

```
待处理(0) → 业务管理员通过(1) → 仓库管理员通过(2) → 人员安排完成(3) → 用户支付定金(4) → 用户支付尾款(5)
                    ↓                           ↓
            业务管理员拒绝(-1)          仓库管理员拒绝(-2)
                    ↓
            用户取消(-3)
```

#### 2.2.3 活动进度查看

- **功能描述**：用户查看自己申请的活动进度
- **显示内容**：
  - 活动基本信息（地点、类型、时间、人数、预算）
  - 当前状态及状态说明
  - 拒绝理由（如被拒绝）
  - 操作按钮（取消活动、支付定金）

#### 2.2.4 活动管理（管理员）

- **功能描述**：管理员查看和管理所有活动
- **功能项**：
  - 查看所有活动申请列表
  - 审批活动（通过/拒绝）
  - 填写拒绝理由
  - 分配负责人

---

### 2.3 人员管理模块

#### 2.3.1 员工管理

- **功能描述**：管理系统员工信息
- **功能项**：
  - 查看所有员工列表
  - 按ID、姓名、职位、城市搜索员工
  - 查看员工详细信息
  - 修改员工职位

#### 2.3.2 员工注册审批

- **功能描述**：审批员工注册申请
- **功能项**：
  - 查看待审批的员工注册列表
  - 批准/拒绝注册申请

#### 2.3.3 人员安排

- **功能描述**：为活动安排工作人员
- **业务规则**：
  - 从员工池中选择合适人员
  - 记录人员与活动的关联关系
  - 更新活动状态为"人员安排完成"

---

### 2.4 物料管理模块

#### 2.4.1 物料信息管理

- **功能描述**：管理系统物料信息
- **功能项**：
  - 添加新物料
  - 查看物料列表（按类别、类型筛选）
  - 修改物料数量
  - 查看物料详情

#### 2.4.2 物料出入库

- **功能描述**：管理物料的出入库操作
- **功能项**：
  - 物料出库（关联活动）
  - 物料入库
  - 查看出入库记录
  - 更新物料租用状态

#### 2.4.3 仓库管理

- **功能描述**：仓库物料管理
- **功能项**：
  - 查看仓库物料库存
  - 按状态筛选物料
  - 审批物料申请

---

### 2.5 薪资管理模块

#### 2.5.1 薪资计算

- **功能描述**：计算员工薪资
- **计算规则**：
  - 根据员工工作记录计算工作天数
  - 根据职位计算日薪
  - 计算应发薪资总额
  - 生成薪资记录

#### 2.5.2 薪资发放

- **功能描述**：管理薪资发放
- **功能项**：
  - 查看待发放薪资列表
  - 单条发放薪资
  - 批量发放薪资
  - 查看已发放薪资记录

#### 2.5.3 个人薪资查询

- **功能描述**：员工查看自己的薪资记录
- **显示内容**：
  - 薪资月份
  - 应发金额
  - 发放状态
  - 发放时间

---

### 2.6 财务管理模块

#### 2.6.1 活动财务统计

- **功能描述**：统计活动的财务收支
- **统计内容**：
  - 活动收入（用户支付的活动费用）
  - 活动支出（物料成本、人员薪资）
  - 活动利润
  - 收入核验功能

#### 2.6.2 报销管理

- **功能描述**：管理费用报销
- **功能项**：
  - 提交报销申请
  - 查看报销记录
  - 审批报销申请
  - 按用户查看报销记录

---

### 2.7 考勤管理模块

#### 2.7.1 工作记录管理

- **功能描述**：记录员工工作情况
- **功能项**：
  - 查看员工工作记录
  - 按状态筛选工作记录
  - 统计已完成/待处理工作项

#### 2.7.2 考勤统计

- **功能描述**：统计员工考勤
- **统计内容**：
  - 月度工作天数
  - 工作时长
  - 考勤状态

---

## 3. 非功能需求

### 3.1 性能需求

- 页面加载时间不超过3秒
- 数据库查询响应时间不超过1秒
- 支持并发用户数不少于50人

### 3.2 安全需求

- 用户密码加密存储
- 敏感操作需要身份验证
- 防止SQL注入攻击
- 使用POST请求传输敏感数据

### 3.3 可用性需求

- 系统可用性达到99.5%
- 支持主流浏览器（Chrome、Firefox、Edge）
- 响应式设计，适配不同屏幕尺寸

### 3.4 数据备份需求

- 定期自动备份数据库
- 支持手动备份和恢复

## 4. 数据需求

### 4.1 核心数据表

#### 用户表（user）

- 用户ID、用户名、密码、角色ID、姓名、电话、性别、状态、创建时间

#### 员工表（employee）

- 员工ID、用户ID、职位、电话、性别、姓名、城市

#### 活动需求表（activity_demand）

- 需求ID、用户ID、活动地点、活动类型、开始时间、结束时间、参与人数、活动预算、活动进度、需求描述、拒绝理由、负责人ID、物料金额、人员金额、活动总价、创建时间

#### 物料表（material）

- 物料ID、物料名称、物料类别、物料类型、数量、单价、状态

#### 薪资记录表（salary_record）

- 记录ID、用户ID、年份、月份、应发金额、发放状态、发放时间

#### 员工工作表（employee_work）

- 工作ID、用户ID、活动ID、工作日期、工作状态

## 7. 附录

### 7.1 活动状态说明

| 状态值 | 状态名称    | 说明        |
| --- | ------- | --------- |
| -3  | 用户取消    | 用户主动取消活动  |
| -2  | 仓库管理员拒绝 | 物料申请被拒绝   |
| -1  | 业务管理员拒绝 | 活动申请被拒绝   |
| 0   | 待处理     | 等待业务管理员审批 |
| 1   | 业务管理员通过 | 活动申请已通过   |
| 2   | 仓库管理员通过 | 物料申请已通过   |
| 3   | 人员安排完成  | 工作人员已安排   |
| 4   | 用户已支付定金 | 定金支付完成    |
| 5   | 用户已支付尾款 | 尾款支付完成    |

### 7.2 角色权限说明

| 角色             | 权限范围                  |
| -------------- | --------------------- |
| 普通用户（roleId=2） | 申请活动、查看活动进度、支付费用、提交报销 |
| 员工（roleId=1）   | 查看工作安排、查看个人薪资、考勤记录    |
| 业务管理员          | 审批活动、安排人员、管理活动进度      |
| 仓库管理员          | 管理物料、审批物料申请           |
| 财务人员           | 薪资管理、财务统计、报销审批        |
| 系统管理员          | 用户管理、权限管理、系统配置        |
